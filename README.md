## 프로젝트 소개

- 카타르 월드컵 16강, 한국 선수들의 빅클럽으로의 이적,활약, 아시안 컵 등으로 인해 최근 한국 축구에 대한 관심이 높아져가고 있습니다.
- build-up은 축구라는 공통의 관심사를 가진 사람들을 위한 플랫폼으로 정보를 공유하고 더 나아가 직접 경기를 뛸 사람들을 모으는 자리를 마련합니다.
